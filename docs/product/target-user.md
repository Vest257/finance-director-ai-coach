# Target User

## Primary User

The initial target user is an ambitious finance professional preparing for or newly operating in a Finance Director role.

Typical profile:

- Senior finance manager, controller, head of finance, FP&A lead, or newly promoted Finance Director.
- Technically capable, but developing executive judgment and broader business leadership.
- Wants practice with scenarios that combine financial analysis, stakeholder management, and decision communication.
- May work in a scale-up, private equity-backed company, mid-market business, or complex operating environment.

## Secondary Users

Later versions may support:

- Aspiring CFOs.
- First-time CFOs.
- Finance leadership coaches.
- Corporate learning and development teams.
- Private equity portfolio talent programs.
- Founders who want to understand finance leadership expectations.

## User Needs

The user needs a safe environment to practice:

- Communicating financial insight clearly.
- Making recommendations with incomplete data.
- Challenging business assumptions.
- Managing cash pressure and runway decisions.
- Preparing board-level narratives.
- Balancing control, growth, and commercial urgency.
- Moving from reporting the numbers to influencing decisions.

## User Frustrations

Common frustrations the product should address:

- Too much finance training is technical but not situational.
- Real leadership feedback is delayed, political, or unavailable.
- Many finance professionals learn stakeholder judgment only after mistakes.
- CFO-level decisions can feel opaque until the user is already accountable for them.

## Initial User Context

The MVP should assume a single learner using a text interface. It should not assume an employer account, cohort, coach dashboard, saved history, or formal credentialing.
